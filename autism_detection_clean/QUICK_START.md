## Troubleshooting

**Script fails immediately?**
1. Check you're in correct directory: `pwd` should show `.../autism_detection_clean`
2. Verify data files exist: `ls X_novelty3.npy` should work
3. Try running one script manually and read error message

**Results look wrong?**
1. Check baseline accuracy ~90.08% (should match previous results)
2. Check novel feature count in importance: should see 10 novel features
3. McNemar p-value: values like 0.06, 0.08, 0.15 are normal (marginal significance)

**PNG file not generated?**
- Install matplotlib: `pip install matplotlib`
- Check disk space has >100MB free
